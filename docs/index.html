About & sources

What you're looking at

Skill = how well each system reproduced past observations: the correlation between its 1993–2016 hindcast and observed precipitation (higher = more trustworthy). Forecast = the current operational outlook for the same 3-month season. Classify each system good/ok/bad from its skill map; the forecasts regroup live so you can weight them accordingly.

Conventions

3-month seasons are picked by issued month + valid trimester; only seasons starting 1–3 months after issue are selectable. The verification fcmonth is the lead to the last month of the window (leadtime + 2). Skill maps are global; the forecast region selector switches the continent embed. Drag to pan, scroll to zoom — synced across all maps of each type.
